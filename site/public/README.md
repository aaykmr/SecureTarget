# Static assets (`public/`)

Files here are copied as-is to the site root when you build. Reference them with paths like `/logo.svg`.

## Logo

| File | Use |
|------|-----|
| `logo.webp` | Navbar & footer (current) |
| `logo.svg` / `logo.png` | Optional alternates |

The UI tries `logo.webp` first, then SVG/PNG. Icon-only logos show beside the **TrustTargets** wordmark.

## Favicons

| File | Purpose |
|------|---------|
| `favicon.ico` | Browser tab |
| `favicon-16x16.png` … `favicon-64x64.png` | PNG favicons |
| `apple-touch-icon.png` | iOS home screen (180×180) |
| `mstile-144x144.png` | Windows tile |
| `site.webmanifest` | PWA manifest (edit icon list here) |
